## 📌 Projects

### 🐶 멍백홈

유실견의 목격 정보를 시간순 지도에 모아
보호자와 구조 활동가가 빠르게 확인할 수 있도록 돕는 서비스입니다.

* 실종 사건 생성 및 공유 링크·QR 제공
* 로그인 없이 목격 정보 제보
* 목격 시간을 기준으로 한 타임핀 지도
* React Native, Supabase, PostgreSQL, PostGIS, Kakao Map API 활용

### 🎙️ SoundLens

한국인 OPIc 학습자의 발음을 분석하고
실행 가능한 피드백을 제공하는 AI 말하기 학습 서비스입니다.

* OPIc 예상 등급 분석
* 발음, 강세, 리듬, 연결 발음 피드백
* Stress Heatmap 및 Sound Map UI 구상
* Next.js와 Supabase 기반 MVP 설계

### ⚾ 야구 구단 추천 서비스

사용자의 응원 성향과 선호도를 분석해
어울리는 KBO 구단을 추천하는 서비스입니다.

* MBTI 형태의 질문 기반 구단 추천
* Django REST Framework와 Vue 활용
* 팀, 선수, 질문, 선택지 데이터 모델링
